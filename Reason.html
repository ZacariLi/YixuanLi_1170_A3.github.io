Why I chose DAO to achieve my goal

Because I think this is a direction of the future development of the network, from the very beginning of the web1 we can only read can not write on it or publish articles, videos, etc., we want to get information can only be conveyed to us through the television station, which is a very centralized performance, because it is they focus on all the information and then integrated and then conveyed to us. Accordingly, our access to information is very single and very slow.

But by web 2.0 we can not only read, we can also write, so gradually there is Facebook, insgram, YouTube, etc., we can comment on it, but also can post videos, no longer rely on the central media, which to some extent removed some centralization.
